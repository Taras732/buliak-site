БУЛЯК — що потрібно для старту

Кілька кроків, щоб магазин запрацював і приймав оплату. Робимо по черзі. Якщо щось незрозуміло — пишіть, разом пройдемо.

📧 Спершу — окрема пошта

Створіть одну окрему поштову скриньку (напр. shop.buliak@gmail.com) і реєструйте на неї все нижче — домен, хостинг, LiqPay тощо. Так усе в одному місці й легко передати.

⚠️ Важливо: під час реєстрації скрізь задавайте звичайний пароль (email + пароль), а не «Увійти через Google». Інакше доступ не передати. Пароль знадобиться розробнику на час налаштування (а за бажанням — і для подальшої підтримки).

1

🌐 Домен (адреса сайту)

Назва, за якою сайт відкривається, напр. buliak.store.

Рекомендую Namecheap моя рекомендація — зручний і недорогий. Аналоги теж є, але з ним простіше.

Namecheap.com →
1-й ріквід $1–5
продовження~$10–30 / рік

Ціни орієнтовні — точні побачите при оформленні.

🔑 Розробнику: логін і пароль від акаунта реєстратора (щоб налаштувати, куди веде домен).
2

🖥️ Хостинг (де працює сайт)

Місце, де сайт живе цілодобово. Для такого магазину потужний сервер не потрібен — достатньо звичайного WordPress-хостингу.

Рекомендую ukraine.com.ua моя рекомендація — український провайдер, є тариф під WordPress.

ukraine.com.ua → Хостинг

Достатньо скромного тарифу:

Тип
WordPress / звичайний хостинг
Диск
від 5–10 ГБ
Технічне
PHP 8, MySQL, SSL
Орієнтовна ціна
~100–200 грн/міс

VPS чи дорогі тарифи не потрібні — провайдер сам дбає про сервер і бекапи.

🔑 Розробнику: доступ до панелі хостингу — щоб встановити й налаштувати сайт.
3

💳 LiqPay (оплата картками)

Через нього клієнти платять на сайті. Реєстрація на liqpay.ua (через Приват24 / ваш ФОП).

liqpay.ua →
  • Зареєструватись і створити компанію/магазин
  • Заповнити дані компанії та банківський рахунок (IBAN) — щоб надходили гроші
  • У розділі Налаштування → API взяти ключі: Public key і Private key

🧪 Для тесту

Ключі з префіксом sandbox_ — перевіряємо оплату без реальних грошей. Потрібні вже зараз.

🚀 Для роботи

Бойові ключі — активуються після верифікації (дані ФОП, рахунок). Підключимо перед запуском.

🔑 Розробнику: ключі Public + Private (спершу sandbox для тесту).
💬 Зробімо разом онлайн — підключусь, підкажу, які поля заповнити, щоб усе одразу запрацювало.
4

🥕 Підписка morkva PRO

Це інструмент, який робить «розумну» оплату для магазину. Закриває одразу кілька задач:

  • Заморозка суми: при замовленні гроші бронюються, а списується фактична сума після зважування (зручно для вагового товару)
  • Чеки (ПРРО): фіскальний чек формується й надсилається клієнту автоматично — окремо налаштовувати не треба
  • Національний кешбек для покупців
morkva LiqPay Extended PRO →
вартість999 грн / рік
🔑 Розробнику: ліцензійний ключ (після покупки) — щоб встановити й увімкнути заморозку та чеки.
5

🚚 Нова Пошта

Щоб клієнт обирав відділення на сайті, а накладні (ТТН) створювались автоматично.

  • У кабінеті НП: Налаштування → Безпека → ключ API (безкоштовно)
  • Часто це працює в зв'язці з morkva — тож, імовірно, окремо нічого не треба, лише ключ
🔑 Розробнику: API-ключ Нової Пошти. Деталі узгодимо на етапі налаштування.

🔐 Доступи для розробника

Щоб усе налаштувати й запустити, потрібен тимчасовий доступ до сервісів вище.

  1. Реєструйте все на окрему пошту, скрізь — email + пароль (не через Google)
  2. Зберіть логіни/паролі в одному документі
  3. Передайте їх безпечно — окремим повідомленням, не у відкритому чаті
  4. Після запуску — на ваш вибір: змінити паролі або лишити доступ для подальшої підтримки
⚠️ LiqPay потребує вашої особистої верифікації (документи ФОП, рахунок) — це проходите ви, а розробник підключає ключі до сайту. Решту можна налаштувати з вашими доступами.
